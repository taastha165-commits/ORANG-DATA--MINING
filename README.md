# ORANG-DATA--MINING

## 📌 Project Overview
This project demonstrates the use of **Orange Data Mining**, an open-source machine learning and data visualization tool, to perform predictive analytics on a financial dataset.

The workflow applies **Logistic Regression** to analyze customer financial information and predict loan default risk. The project showcases the complete machine learning pipeline, from data import and feature selection to model training and evaluation.

---

## 🎯 Objective
The objective of this project is to:

- Import and preprocess financial data.
- Select relevant features for analysis.
- Build a predictive model using Logistic Regression.
- Evaluate model performance using testing and scoring metrics.
- Demonstrate machine learning concepts using Orange Data Mining.

## 📊 Dataset Features

The dataset contains customer financial and demographic information such as:

- Age
- Annual Income
- Loan Amount
- Credit Score
- Employment Years
- Existing Loans
- Credit Card Debt
- Other financial indicators

These variables are used to predict whether a customer is likely to default on a loan.

---

## 🔄 Workflow

The Orange workflow consists of the following components:

### 1. File
Imports the CSV dataset into Orange.

### 2. Select Columns
Chooses the relevant features required for model training.

### 3. Logistic Regression
Builds a classification model to predict loan default probability.

### 4. Test and Score
Evaluates the model performance using standard classification metrics.

---

## 📈 Machine Learning Model

**Algorithm Used:** Logistic Regression

Logistic Regression is a supervised machine learning algorithm commonly used for binary classification problems. In this project, it predicts whether a loan applicant is likely to default based on financial characteristics.
